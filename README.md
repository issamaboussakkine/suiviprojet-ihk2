🧾 Suivi de Projet IHK

Application web complète de gestion de projets développée avec :

⚙️ Backend : Spring Boot
🎨 Frontend : React (Vite + Tailwind)
🔐 Sécurité : JWT + Spring Security
🗄️ Base de données : MySQL
